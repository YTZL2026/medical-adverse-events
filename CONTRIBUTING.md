# 贡献指南

感谢你考虑为医疗不良事件报告系统做出贡献！

## 如何参与

### 报告 Bug
1. 在 [Issues](https://github.com/YTZL2026/medical-adverse-events/issues) 页面新建 Issue
2. 描述：做了什么、期望看到什么、实际看到什么
3. 附上截图或错误信息更佳

### 提新功能
1. 先搜搜有没有人提过类似的
2. 新建 Issue，用 `[Feature Request]` 做标题
3. 描述使用场景：谁、什么场景下、需要什么功能

### 提交 PR
1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/my-feature`
3. 修改代码 + 测试
4. 提交：`git commit -m "feat: 添加XXX功能"`
5. 推送：`git push origin feature/my-feature`
6. 发起 Pull Request

## 代码风格

- HTML/CSS：2空格缩进，语义化标签
- JS：ES6+，驼峰命名
- Python：PEP8，4空格缩进
- 提交信息用中文描述清楚改动

## 优先需要的贡献

- 🐛 边界情况测试
- 📝 文档完善和案例补充
- 🌐 英文翻译
