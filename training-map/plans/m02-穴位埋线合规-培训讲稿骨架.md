## 三、核心讲稿要点（按章节）

### Part 1：2025重大变更（2分钟，⚠️ 最重要的开场）
> 这是整个培训最核心的内容，必须先讲清楚规则变了

**旧模式（已废止）**：
- ✘ 按穴位个数收费 → 做8个穴收8次费
- ✘ 埋线针/线体单独收费 → 治疗费100+材料费50
- ✘ 埋线与针刺叠加收费 → 同一天收两份钱

**新模式（强制执行）**：
- ✓ 统一按"次·日"收费 → 不管做几个穴，一天只收一次
- ✓ 针+线计入基本物耗 → 不再另收材料费
- ✓ 按最高项目标准计费 → 同一部位不叠加

**法规依据**：辽医保发〔2025〕36号——辽宁省中医特殊疗法类价格项目通知

**违规后果**：追回全部违规基金 + 约谈负责人 + 责令整改 + 公开通报

### Part 3：收费合规详解（2分钟）
**计价单位**：
- 按"次·日"收费，不是按"穴·次"
- 单日仅收费1次
- 编码必须与辽宁省最新价格项目映射一致

**耗材**：
- 埋线针→含在治疗费中
- 可吸收缝线→含在基本物耗中
- 局麻药→含在项目价格中

**叠加收费禁止**：
- 埋线+常规针刺→只收一种
- 埋线+温针/电针→禁止叠加

**案例警示**：某门诊部按8个穴收费+材料另收→被定性"超标准收费+重复收费"

### Part 5：院感无菌（2分钟）
> 这是飞行检查的高频扣分项，抽查必看

六步洗手+手消毒 → 皮肤消毒≥5cm×5cm，由内向外2遍 → 消毒棉球一穴一换 → 一次性针具4小时内使用 → 利器盒规范投放 → 紫外线消毒每日有记录

### Part 9：迎检话术（3分钟）
> 六类场景，每类一句话原则+标准话术示范

见 MD 源文档第十一章的六段场景话术，每段1分钟。

## 四、考核设计

### 理论考核（12道题）
1. 穴位埋线收费的基本单位是？→ 次·日（不是穴·次）
2. 埋线针和线体的费用如何处理？→ 含在治疗费中，不另收
3. 皮肤消毒面积要求？→ ≥5cm×5cm
4. 一次性针具打开后几小时内使用？→ 4小时
5. 六步洗手法的顺序？（实操考核用）
6. 知情同意书必须包含哪几个要素？→ 适应症+操作过程+可能风险+术后注意+不良反应+替代方案

### 实操考核
- 六步洗手法 + 消毒操作（皮肤消毒面积达标+棉球一穴一换）
- 知情同意书完整签署（模拟：核对6要素是否齐全）
